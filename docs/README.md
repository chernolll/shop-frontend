# 🎯 项目背景（简要）

这是一个电商 BD 管理系统，核心流程为：

任务（task_main）→ 分配BD（task_bd_relation）→ 达人筹备（kol_bd_prepare）→ SOP执行（task_sop）→ 视频产出（video）→ GMV统计

系统基于 Vben Admin 开发，前端为 Vue3 + TypeScript。
