- 先读 `docs/README.md` 和 `docs/crrent-state.md`

- 保持现有技术栈：
  - Vue 3
  - TypeScript
  - Vben Admin
  - vxe-table
- 优先复用现有组件、路由模式、store 和 API 组织方式

## 输出要求

- 简洁

- 先讲关键判断，再讲实现
- 涉及业务流程时，以后端状态和表结构为准，不靠猜
